Concha del mar Project

Reservation information management C# system

  • Home

Summary

Design the catalogs and database structure, as well as the graphic part of the reservation information management system in order to optimize data security and handling.

Problem

The Camping & RV Park had a manual booking record. The information remained in physical documents that were in danger of being easily lost or destroyed when consulted or when a contingency occurred within the establishment.

Description

The technologies and methodologies implemented in this software are:

  • C#
  • Layered architecture
  • SCRUM
  • SQL
  • SQL Report Services
  • POO
  • WinForm
