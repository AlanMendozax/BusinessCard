TecNM Project

Grade reporting web platform

  • Home

Summary

It seeks to streamline and facilitate the grade reporting process for teachers through the development of a platform with the help of models and technologies for web development. It has a database in which information is stored and queries are made. It is hosted in the cloud to access its services remotely as long as the user has internet or bandwidth service to access the web application.

Problem

The National Technological Institute of Mexico requested software from the students of the systems engineering career which presented three modules, one of them is the capture of qualification formats which require compliance with the ISO 14000 and 14001 standard.

Description

The technologies and methodologies implemented in this software are:

  • Bootstrap
  • CSS
  • HTML
  • SCRUM
  • PHP
  • React js
  • Three-tier client/server architecture
